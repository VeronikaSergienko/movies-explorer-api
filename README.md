**Как установить и запустить проект:**  

------ 
1. Клонировать репозиторий:    
``` git clone https://github.com/VeronikaSergienko/movies-explorer-api.git ```
2. Установить зависимости:  
``` npm install ```
3. Запустить проект на localhost:  
``` npm start ```
